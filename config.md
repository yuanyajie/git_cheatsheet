# Custom configuration

-   Initial config
    
        git config --global user.name "Yajie Yuan"
    
        git config --global user.email <email>
    
        git config --global core.editor vim
    
        git config --global color.ui auto

-   Checking your settings

        git config --list

-   Getting help

	    git help config
	
	    git config --help

	    man git-config
